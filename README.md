# 📝 ToDayDo

ToDayDo is a simple and clean To-Do application built using Flutter.  
The app helps users organize daily tasks and manage their productivity easily.

---

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Dynamic task list
- State management using Provider
- Clean and responsive UI

---

## 🛠️ Built With

- Flutter
- Dart
- Provider

---

## 📂 Project Structure

lib/
- models/
- screens/
- widgets/
- main.dart
